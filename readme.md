# 교육명 : Python과 딥러닝을 활용한 시계열 자료 분석 과정

## 교육 목적

* Python과 딥러닝을 활용하여
  **시계열(Time Series) 데이터의 처리, 분석, 예측 방법을 체계적으로 학습**
* 통계 기반 기법부터 딥러닝(LSTM 등)까지
  **전통적 방법 + 최신 AI 모델의 차이와 활용법을 이해**
* 실제 데이터(주식, 에너지, 행동 인식 등)를 활용한
  **실습 중심 예측 모델 구축 능력 배양**

> *“시계열 데이터를 분석하고, 예측 모델을 직접 만들 수 있는 실무 역량 확보”* 가 핵심 목표입니다.

## 3️⃣  일자별 교육 내용 상세 설명 (1일 5시간 기준)

## 📅 1일차: 개발 환경 구축 및 탐색적 데이터 시각화(EDA)

### 1일차 - 학습 목표

1. 파이썬 기본 문법과 디버깅 도구 사용법을 익혀 안정적인 개발 환경을 구축한다.
2. 다양한 시각화 라이브러리를 통해 시계열 데이터의 패턴과 변수 간 상관관계를 분석하는 EDA 기술을 습득한다.

### 1일차 - 주요 교육 내용

* 환경 설정 및 디버깅 및 기초 용어
* 기본 시계열 시각화
* 통계적 시각화 및 상관분석

---

## 📅 2일차: 시계열 분해 및 딥러닝 학습 파이프라인

### 2일차 - 학습 목표

1. 이동평균 기법을 활용하여 데이터의 노이즈를 제거하고 추세를 분석하는 시계열 분해 원리를 이해한다.
2. 딥러닝 모델 개발의 전체 프로세스를 익히고 RNN의 순환 구조가 시계열에 어떻게 적용되는지 학습한다.

### 2일차 - 주요 교육 내용

* 시계열 분해와 평활화
* 딥러닝 개발 단계 및 기초 모델링
* RNN(순환신경망) 원리

## 📅 3일차: 데이터 전처리 및 LSTM 예측 모델

### 3일차 - 학습 목표

1. 시계열 데이터를 딥러닝이 이해하는 3D Window 형태로 변환하고 결측치/정규화를 처리하는 실무 기술을 습득한다.
2. LSTM의 게이트 시스템을 이해하고, 단일 변수 시계열 데이터를 이용한 미래 값 예측 모델을 구현한다.

### 3일차 - 주요 교육 내용

* 시계열 전처리 및 지도학습 변환
* LSTM 기초 및 단변수 예측
* 하드웨어 가속 설정

## 📅 4일차: 다변수 예측 및 고급 아키텍처

### 4일차 - 학습 목표

1. 다차원 독립 변수를 활용한 예측 모델과 Stacked LSTM을 구축하여 예측 성능을 고도화한다.
2. Attention 메커니즘을 적용하여 특정 시점의 중요도를 계산하는 고급 딥러닝 기법을 실습한다.

### 4일차 - 주요 교육 내용

* 다변수 모델링
* 다층 LSTM 모델링
* 딥러닝 모델 실무
* Attention 메커니즘

## 📅 5일차: 도메인 특화 실무 및 시계열 이상 탐지

### 5일차 - 학습 목표

1. 주식 시장 및 행동 인식 센서 데이터를 활용하여 각 도메인 특성에 맞는 시계열 분류와 예측을 수행한다.
2. 비지도 학습 기반의 이상치 탐지 기법을 익혀 산업 현장의 사고 예방 및 비정상 감지 시스템을 구축한다.

### 5일차 - 주요 교육 내용

* 에너지 사용량 예측
* 도메인 실무 (금융 & 센서)
* 시계열 이상치 탐지(Anomaly Detection)
* 실전 추론 프로그램 구현 및 교육 마무리
